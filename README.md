# Documentação SOEA

Repositório de planejamento e execução do Sistema Operacional de Estudos Adaptativo.

## Como iniciar

1. Clone ou baixe este repositório.
2. Abra a pasta no Codex do aplicativo ChatGPT para Windows.
3. Ative Computer Use.
4. Conecte a extensão do Chrome.
5. Execute somente o Prompt 00.
6. Siga o processo descrito em `FluxoCodex.md`.

## Ordem das etapas

1. Prompt 00 — ambiente;
2. Prompt 01 — análise;
3. Prompt 02 — estrutura;
4. Prompt 03 — motor;
5. Prompt 04 — finalização.

Nunca execute todas as fases de uma vez.
